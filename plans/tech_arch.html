System architecture

Key data flows

Guest request flow

1Guest messages in linked group chat (any platform, any language)
2COZMO reads → LM Studio (Gemma 4) classifies intent: request / question / cancel
3Service request → PATCH Hostfully note → JANDI alert fires to property topic
4Translation via OpenAI gpt-4o-mini if platform needs it (WeChat, LINE)
5Staff sees JANDI alert → acts. Never monitors raw chat.

New booking flow

1Hostfully fires NEW_BOOKING → Cloudflare Tunnel → Express :3001
2Fetch lead + property from Hostfully API. Skip if not BOOKED / PAID_IN_FULL.
3Alert staff via JANDI + Telegram with platform availability (WA / Telegram)
4Create WA group via Evolution API → pull team from Sheets → auto-link groupId → lead_uid
5LLM generates welcome → send 4-message sequence. Promote booked guest to group admin.

New group member flow

1Booked guest is given WA group admin on group creation
2Admin guest adds family / travel companions to the group
3COZMO detects group_participants_update join event
4Lookup lead_uid from group-leads.json, fetch property info from Hostfully
5Send info to new member only. No group spam.

Expense + checkout flow

1Staff types /exp 50000 Airport van in linked group (any platform)
2COZMO appends to Google Sheets expenses tab — lead_uid, item, amount, VAT, settled=false
321:00 KST night before checkout → auto-query all unsettled expenses for that lead
4Calculate total + VAT variants (10% / 14.5%) with cash / WISE / bank transfer options
5Send formatted bill to guest group → auto-settle rows. 07:00 KST sends final bill if new items added after 9PM.
